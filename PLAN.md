# CatchGoose 复刻规划（可迭代）

目标：在 Flutter + Flame 中复刻核心玩法与手感，做 2D/2.5D 视觉效果的移动端版本。

## 范围与假设
- 玩法：点击“可见/无遮挡”物品加入槽位，三消即消，槽位 7 格。
- 目标：倒计时内清空或完成目标即可过关。
- 道具：暂不设计，仅保留占位 UI。
- 美术：不使用原作素材，使用自有或占位素材。

## 里程碑
1. **M1 可玩原型**
   - 物品堆叠展示（2D sprite + 深度排序）。
   - 点击可见物品 -> 入槽位。
   - 槽位 7 格，3 个相同自动消除。
   - 倒计时与失败条件（槽位满/时间到）。

2. **M2 视觉与手感接近**
   - 物品轻微弹起、飞入槽位、三消消散动画。
   - 遮挡判断更精确（层级/遮挡区域）。
   - UI 结构靠近原作（顶部进度、底部道具栏）。

3. **M3 数据驱动关卡**
   - JSON 关卡配置。
   - 难度曲线与物品分布。

## 技术方案
- 引擎：Flame（2D），通过 `priority` 做深度排序。
- 物品层：`ItemComponent`（Sprite + hit-test）。
- 遮挡规则：点击时只允许当前点击点“最上层”物品。
- 槽位栏：固定 7 格容器，维护物品队列。
- 数据：`assets/levels/*.json` + `assets/sprites/*`。

## 资源与目录（建议）
- `assets/sprites/` 物品与 UI 图片
- `assets/levels/` 关卡配置
- `lib/game/` Flame 游戏核心
- `lib/ui/` Flutter UI（菜单/关卡选择）
